Projects

Information on specific projects:

JAGS

Matt has been contributing to development of Just Another Gibbs Sampler (JAGS; general-purpose software for fitting models using MCMC not unlike WinBUGS/OpenBUGS) for around 5 years. He also builds and maintins the macOS binary distribution of the software, and an R package runjags that is designed to facilitate using JAGS from within R.

If you want to learn how to use JAGS and runjags to fit models to your data you could look at the ABME course that is run annually in Glasgow in collaboration with PR statistics.

Analysis of FECRT data

Matt has been working on methods of analysing FECRT data for almost 15 years and has set up a website with links to tools etc here.
